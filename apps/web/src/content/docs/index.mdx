## 快速开始

1. 在控制台创建 **API Key**（外部 API 需 Starter 及以上套餐）。
2. 读 **[外部 API](/docs/external-api)** 的「鉴权」与「通用约定」。
3. 图像走 `POST /v1/images/generations`；**视频是长任务、默认走异步**，按 [视频 · 异步流程](/docs/external-api#异步流程推荐) 提交后轮询 `GET /v1/videos/{id}` 或用 `callback_url` 回调。
